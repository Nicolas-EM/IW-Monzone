MONZONE

La aplicación web para dividir gastos

Introducción

Monzone es la solución perfecta para dividir gastos entre amigos, familiares, o compañeros de piso. Con esta aplicación web se pueden crear grupos de gastos, agregar compras, y dejar que se calculen las cantidades que cada persona debe. Además, Monzone es la única plataforma que te deja configurar un presupuesto y hacer copias de seguridad totalmente privadas.

Roles

Al crear un grupo, automáticamente se asigna el rol de administrador de grupo al creador.

Cualquier otro miembro del grupo se considera un usuario no administrador.

Tareas

Ver Estadísticas

  • Este producto se basa en poder ver cuánto dinero cada usuario debe, y a quién se lo debería pagar para reducir el número de pagos que tiene que hacer. Las estadísticas muestran esto, además de un análisis de los gastos (ej. cantidad por categoría).

Crear Grupo

  • Cualquier usuario puede crear un grupo e invitar a otros usuarios para empezar a dividir los gastos.

Añadir / Editar / Eliminar Miembros

  • Un administrador de grupo puede añadir a miembros nuevos, eliminar a miembros que ya no deban dinero o cambiar el rol de un miembro.

Añadir / Editar / Eliminar Gastos

  • Cualquier usuario puede añadir un gasto, indicando cuantía, categoría y cómo se divide el gasto, si equitativamente o no.

  • Un administrador de grupo puede editar los gastos de cualquier otro miembro, mientras que un miembro normal solo puede editar los gastos que haya creado.

  • Un administrador puede eliminar cualquier gasto.

    Al hacer cualquiera de estas operaciones, se notifica a todos los usuarios involucrados.

Configurar Presupuesto Individual / Grupal

  • Se notifica cuando se haya gastado un 50%, 75%, 90% y 100% del presupuesto.

  • En el caso del presupuesto individual, se notifica a ese usuario. En el presupuesto grupal, se notifica a todos los usuarios del grupo.
